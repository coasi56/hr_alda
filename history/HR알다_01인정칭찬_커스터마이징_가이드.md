## 1. 반드시 변경해야 하는 항목

### 1-1. 환경변수 (backend/.env 및 Render 설정)

| 변수명 | 설명 | 획득 방법 |
|--------|------|-----------|
| `SLACK_BOT_TOKEN` | Slack Bot OAuth Token | api.slack.com → OAuth & Permissions → Bot User OAuth Token |
| `SLACK_SIGNING_SECRET` | Slack 서명 검증 키 | api.slack.com → Basic Information → Signing Secret |
| `JWT_SECRET` | 인증 토큰 서명 키 (32자 이상) | `node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"` |
| `ADMIN_EMAIL` | 관리자 대시보드 로그인 이메일 | 직접 설정 (예: hr@yourcompany.com) |
| `ADMIN_PASSWORD` | 관리자 대시보드 로그인 비밀번호 | 직접 설정 (영문+숫자+특수문자 조합) |
| `DATABASE_URL` | PostgreSQL 연결 URL | Render PostgreSQL → Internal Database URL |

### 1-2. Vercel 환경변수

```
VITE_API_URL = https://[본인의 Render 백엔드 URL]
```

### 1-3. CORS 허용 도메인 (backend/src/app.js)

```javascript
app.use(cors({
  origin: [
    'http://localhost:5173',
    'https://[본인의 Vercel URL]',  // ← 이 줄 변경
  ],
}));
```

## 3. AI 활용 프롬프트 템플릿

Claude, ChatGPT, Gemini 등 어떤 AI에게도 아래 프롬프트를 사용하면 효과적으로 도움을 받을 수 있습니다.

### 3-1. 프로젝트 소개 (새 대화 시작 시 사용)

```
나는 HR알다라는 Slack 기반 HR 자동화 시스템을 운영하고 있어.

기술 스택:
- 백엔드: Node.js + Express + Slack Bolt for JS
- 프론트엔드: React + Vite
- DB: PostgreSQL (Render)
- 백엔드 배포: Render.com
- 프론트엔드 배포: Vercel
- GitHub: https://github.com/[내 계정]/hr_alda

주요 기능:
1. Slack 이모지(:알다-칭찬:, :알다-신뢰:, :알다-주도성:, :알다-원팀:) 기반 칭찬 시스템
2. 주간 10개 한도 관리
3. 슬래시 커맨드: /alda_left, /alda_check, /alda_board
4. React 관리자 대시보드
5. PostgreSQL DB 영구 저장

나는 비개발자야. 쉽고 단계별로 설명해줘.
```

## 4. 프로젝트 파일 구조

| 파일 경로 | 역할 |
|-----------|------|
| `backend/src/app.js` | Express 앱 설정, CORS, 라우트 등록 |
| `backend/src/server.js` | 서버 시작, DB 연결, Slack 초기화 |
| `backend/src/config/slack.js` | Slack Bolt 앱 초기화 |
| `backend/src/config/database.js` | PostgreSQL 연결 및 테이블 생성 |
| `backend/src/services/aldaService.js` | 핵심 비즈니스 로직 (이모지 집계, 한도 관리) |
| `backend/src/slack/listeners/reactionAdded.js` | 이모지 추가 감지 및 처리 |
| `backend/src/slack/listeners/reactionRemoved.js` | 이모지 제거 감지 및 처리 |
| `backend/src/slack/commands/aldaLeft.js` | /alda_left 커맨드 처리 |
| `backend/src/slack/commands/aldaCheck.js` | /alda_check 커맨드 처리 |
| `backend/src/slack/commands/aldaBoard.js` | /alda_board 커맨드 처리 |
| `backend/src/routes/auth.js` | 관리자 로그인 API |
| `backend/src/routes/alda.js` | 대시보드 데이터 API |
| `backend/src/middleware/authMiddleware.js` | JWT 인증 미들웨어 |
| `frontend/src/pages/Login.jsx` | 로그인 페이지 |
| `frontend/src/pages/AldaPage.jsx` | 대시보드 메인 페이지 |
| `frontend/src/components/alda/AldaTable.jsx` | 리액션 이력 테이블 |
| `frontend/src/components/alda/AldaChart.jsx` | 이모지별 현황 차트 |
| `frontend/src/api/client.js` | API 호출 클라이언트 |
